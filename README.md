# vue-cource
## 项目结构
- `src/`  开发文件夹
- `src/api/`  API文件夹
- `src/assets/` 资源文件夹
- `src/assets/font/`  字体文件
- `src/assets/img/`  图片
- `src/config/` 项目配置文件
- `src/directive/`   vue的自定义指令
- `src/lib/` js方法
- `src/lib/util.js`  与业务相关的js方法
- `src/lib/tool.js`  与业务不相关的js方法
- `src/router` 路由文件夹
- `src/store/` 全局状态管理
- `src/components/` 组件文件夹
- `src/views/` 页面
- `src/mock/` 模拟数据
